# Legal Documents

This repository contains legal documents used across Chihipiro software applications.

---

## 内容

本リポジトリには以下の法務ドキュメントが含まれます。

- [Common End User License Agreement (Common EULA)](common\Common-EULA.md)
- Product-specific additional terms（製品ごとの追加利用条件）

## 目的

BOOTHなどで配布・販売されるソフトウェアにおける利用条件・制限・免責事項を明確にすることを目的としています。

## 構成

- [Common-EULA.txt (Official)](common\Common-EULA.txt)  

  → Applies to all applications（すべてのアプリに適用される共通契約）

- Product-specific terms  

  → Additional conditions per application（各アプリケーション固有の追加条件）

---

## 注意事項

各アプリケーションには、Common EULAに加えて個別の追加条件が適用される場合があります。
